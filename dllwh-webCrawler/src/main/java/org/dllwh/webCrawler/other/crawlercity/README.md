模块说明
---

* **crawlerCity:爬取行政区域(省市县)**

> + [2015年统计用区划代码和城乡划分代码(截止2015年09月30日)](http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2015/index.html)

> +    《统计用区划代码》的结构采用五层12位数字表示


> > + `1、2位表示省级码段`
> > + `3、4位表示地级码段`
> > + `5、6位表示县级码段`
> > + `7～9位表示乡级码段`
> > + `10～12位表示村级码段`

> + 《统计用区划代码》城乡分类代码

> > + `111表示主城区`
> > + `112表示城乡结合区`
> > + `121表示镇中心区`
> > + `122表示镇乡结合区`
> > + `123表示特殊区域`
> > + `210表示乡中心区`
> > + `220表示村庄`
> > + `城乡分类代码第1位为1，表示城镇；第1位为2，表示乡村`
